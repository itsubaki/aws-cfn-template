# Immutable system on aws

## 解決したいもの

1. AWSアカウント作成後にシステムを1ボタンで構築したい
2. システム全体のバージョン管理したい
 + 開発環境で任意のバージョンを構築したい
 + 本番環境で複数のバージョンを同時に構築したい


# 基本

## レイヤー

+ システムを以下のレイヤーに分けて考える
 - インフラ（リソース）
 - ミドルウェア(運用・監視ツール含む)
 - アプリケーション
