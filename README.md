# Immutable system on aws

## 何を解決するか?

1. アカウント作成後にシステムを1ボタンで構築したい
2. システム全体のバージョン管理したい
 + 開発環境で任意のバージョンを構築したい
 + 本番環境で複数のバージョンを同時に構築したい

## Immutableにできないもの

1. アプリケーションデータ(データベース)

# 基本

1. システムは以下に分解できる
 - インフラ（リソース）
 - ミドルウェア(運用・監視ツール含む)
 - アプリケーション
